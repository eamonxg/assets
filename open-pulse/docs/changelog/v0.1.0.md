# v0.1.0

Initial release of **OpenPulse**, a lightweight network monitoring service for OpenWrt.

### Features
- **System Monitoring**: CPU, memory, temperature, and uptime.
- **Network Status**: WAN status, IP, DNS, and traffic statistics.
- **Device Discovery**: Real-time list of connected devices (DHCP + ARP).
- **DNS Analytics**: Query stats and top domains (requires dnsmasq log-queries).
- **Log Access**: System logs via `logread`.
- **SSE Stream**: Real-time metrics push for the mobile app.
- **Secure by Design**: Binds to LAN only, 128-bit random token auth, and brute-force lockout.

### Requirements
- OpenWrt 23.05 or newer.
- `ucode`, `uhttpd`, `uhttpd-mod-ucode`.
